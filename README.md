# TypeScript Labs by Camel Djoulako

Professional TypeScript lab showcasing my understanding of modern JavaScript, strong typing, and clean architecture principles. Part of my software engineering portfolio.

---

## About

This repository contains exercises and mini-projects developed during my journey through the Discover TypeScript course.  
It covers fundamentals to advanced topics of TypeScript with a focus on clean, modular, and maintainable code.

---

## Usage

Clone the repository:

```bash
git clone https://github.com/CamelDjoulako/typescript-labs-by-cameldjoulako.git
```

## Navigate to a folder and install dependencies

```bash
cd typescript-labs-by-cameldjoulako
npm install
```
